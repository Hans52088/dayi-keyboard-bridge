# ⌨️ 大易實體鍵盤轉接器 - iPad 離線完整版 (Dayi Bridge Offline)

這是一個專為 iPad（以及其他移動設備）設計的 **100% 離線型** 大易輸入法實體鍵盤轉接程式。

## 🌟 核心特色

- **完全離線**：所有字元資料（約 18,000+ 字）均直接內嵌於本地檔案，無需任何網路連接。
- **極速回應**：即按即顯，無網路延遲，提供最接近原生輸入法的流暢體驗。
- **隱私安全**：所有輸入過程均在您的瀏覽器本地處理，不經過任何伺服器。
- **iPad 優化**：完美匹配實體鍵盤的按鍵邏輯，特別適合使用鍵盤套的 iPad 用戶。
- **現代介面**：採用磨砂玻璃質感 (Glassmorphism) 設計，支援動態背景。

## 💡 快速使用指南

這是一個「打字轉接橋樑」，您的操作流程如下：

1. **大易輸入**：在頁面中使用實體鍵盤的英文模式打字，程式會根據大易字根將其組合為中文字。
2. **一鍵複製**：打完整句話後，按下 **Enter** 鍵。
3. **自動處理**：系統會自動將整段文字複製到剪貼簿，並清空目前的輸入區。
4. **切換貼上**：切換到您想要輸入文字的 App（如備忘錄、Line、郵件等），直接 **「貼上」** 即可。

透過這種方式，您可以突破 iPad 系統限制，享受實體鍵盤大易輸入的極速快感。

## 🛠️ iPad 安裝方式

> ⚠️ **重要說明**：iPad「檔案」App 的預覽功能（Quick Look）**不支援 JavaScript**，
> 因此不能直接在「檔案」App 中點開 HTML 檔案使用。請依照以下方式安裝：

### 方法一：GitHub Pages + 加入主畫面（推薦 ✅）

1. 在 iPad 的 **Safari** 中開啟此頁面的線上連結（例如 GitHub Pages 網址）。
2. 點擊 Safari 底部的 **分享按鈕** (⬆️ 方框+箭頭)。
3. 向下滑動找到 **「加入主畫面」** 並點擊。
4. 點擊 **「新增」**。
5. 主畫面上就會出現大易轉接器的圖示，**之後即可完全離線使用**！

### 方法二：使用第三方瀏覽器 App

1. 在 iPad 上安裝 **Documents by Readdle**（免費）。
2. 將 `Dayi-ActualKB-transfer(Offline).html` 放入 iPad 的檔案資料夾。
3. 用 Documents App 開啟該 HTML 檔案，其內建瀏覽器完整支援 JavaScript。

## ⌨️ 按鍵說明

| 按鍵 | 功能 |
|------|------|
| `A-Z / 0-9` | 輸入大易字根 |
| `Space` | 選第 1 個候選字 |
| `'` | 選第 2 個候選字 |
| `[` | 選第 3 個候選字 |
| `]` | 選第 4 個候選字 |
| `-` | 選第 5 個候選字 |
| `\` | 選第 6 個候選字 |
| `=` | 進入/退出標點模式 |
| `Enter` | 複製全部文字並清空 |
| `Backspace` | 刪除 |
