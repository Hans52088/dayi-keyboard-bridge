# 🌐 大易實體鍵盤轉接器 - 半離線版 (Dayi Bridge Semi-Offline)

這是一個輕量化的 **半離線型** 大易輸入法實體鍵盤轉接程式，透過動態載入線上標準 `.cin` 字表來實現輸入功能。

## 🌟 核心特色

- **輕量化單檔**：主程式體積小，無需額外附帶龐大的 JS 字典檔。
- **動態字典載入**：首次或是需要更新時，可一鍵從 GitHub (cin-tables) 下載最新的大易四碼字表。
- **本地快取 (LocalStorage)**：成功下載後，字典會快取在您的瀏覽器記憶體中，之後即便沒有網路也能正常使用（除非清除瀏覽器資料）。
- **完全標準**：解析標準的 `.cin` (Chartdef) 格式，確保字碼正確。
- **現代介面**：同樣採用極具現代感的磨砂玻璃透明設計，支援 iPad 全螢幕使用。

## 💡 快速使用指南

這是一個「打字轉接橋樑」，您的操作流程如下：

1. **大易輸入**：在頁面中使用實體鍵盤的英文模式打字，程式會根據大易字根將其組合為中文字。
2. **一鍵複製**：打完整句話後，按下 **Enter** 鍵。
3. **自動處理**：系統會自動將整段文字複製到剪貼簿，並清空目前的輸入區。
4. **切換貼上**：切換到您想要輸入文字的 App（如備忘錄、Line、郵件等），直接 **「貼上」** 即可。

透過這種方式，您可以突破 iPad 系統限制，享受實體鍵盤大易輸入的極速快感。

## 🛠️ 安裝與使用

1. **檔案下載或線上直接使用**：
   - 線上使用 https://hans52088.github.io/web/Dayi-ActualKB-transfer(Online).html
   - 下載 `Dayi-ActualKB-transfer(Online).html`。
2. **首次連接**：
   - 開啟頁面後，點擊右下角的 **「📥 載入大易字庫」**。
   - 程式會從 GitHub 抓取資料並快取。
3. **離線使用**：
   - 完成首次載入後，在此網頁未關閉下此功能仍可正常操作(若關閉網頁或重開機需重新載入大易字庫)。
4. **操作邏輯**：
   - 支援 A-Z/0-9 大易字根及標點符號模式。
   - 選字按鍵與功能按鍵邏輯與完整版一致。
